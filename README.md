# rembg - 対象物自動切り抜きツール

このアプリケーションは、rembgを使用して画像の背景を自動的に除去するGUIツールです。

## 機能

- **画像読み込み**: JPG、PNG、BMP形式をサポート
- **マウス選択**: ドラッグで領域を指定（オプション）
- **背景除去**: rembgを使用して自動的に背景を除去
- **結果保存**: PNG形式で透明背景付きで保存

## セットアップ

### 1. Python環境の準備

```bash
# 仮想環境を作成（オプション）
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

### 2. 依存パッケージをインストール

```bash
pip install -r requirements.txt
```

- GPUを使いたい場合:

```bash
pip install -r requirements-gpu.txt
```

## 使用方法

### アプリケーションの起動

```bash
python src/app.py
```

### 操作手順

1. **「画像を開く」をクリック**
   - 処理したい画像ファイルを選択します

2. **対象領域を選択（オプション）**
   - マウスでドラッグして、処理したい領域を指定します
   - 領域の指定なしでも全体に対して背景除去が行われます

3. **「背景を除去」をクリック**
   - rembgが背景を自動的に除去します
   - 処理には数秒から数十秒かかる場合があります

4. **「結果を保存」をクリック**
   - 透明背景付きのPNG形式で保存します
   - デフォルトでは `output` フォルダに保存されます

## 主なボタン

| ボタン | 機能 |
|--------|------|
| 画像を開く | 画像ファイルを選択 |
| 選択をクリア | マウスで指定した領域選択をクリア |
| 背景を除去 | rembgで背景を除去（選択領域がある場合は領域内のみ） |
| 結果を保存 | 処理済み画像をPNG形式で保存 |
| リセット | すべての処理をリセット |

## トラブルシューティング

### 画像が表示されない
- ファイル形式が正しいか確認してください
- OpenCVがサポートする形式: JPG, PNG, BMP

### 背景除去が遅い
- 大きな画像を使用している場合、処理時間が増加します
- GPUを使うには、`requirements-gpu.txt` を使ってインストールし、NVIDIA GPUと対応するCUDAドライバーが必要です

### GPUが動作しない場合
- `rembg[gpu]` のインストールが必要です
- `onnxruntime-gpu` のインストールに失敗した場合はCUDAバージョンを確認してください
- CPUのみ利用する場合は `requirements.txt` を使います

### メモリ不足エラー
- 非常に大きな画像を使用している場合、画像をリサイズしてから処理してください
